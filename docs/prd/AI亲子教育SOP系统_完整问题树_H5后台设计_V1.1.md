# AI亲子教育SOP系统：完整问题树 + H5前端 + 后台内容管理平台 V1.1

> 项目定位：作为《AI时代家庭教育升级课》5天训练营的核心交付物。  
> 使用场景：家长在手机H5里按场景查找问题，点到最后得到解决方案、操作SOP、AI提示词、沟通话术；运营团队在电脑端后台持续新增、修改、上下架内容。

---

## 0. 总体产品定位

### 0.1 一句话定位

**AI亲子教育SOP系统 = 家长遇到孩子问题时，可以立刻查、立刻问、立刻复制提示词、立刻执行的家庭教育行动工具。**

它不是简单的资料库，也不是单纯的提示词合集，而是：

- 家庭教育问题解决库
- AI提示词操作库
- 家长沟通话术库
- 学科辅导SOP库
- 孩子成长路径库
- 课程资料交付系统
- 后续可升级为AI家庭教育Agent的内容底座

### 0.2 核心用户

1. 小学生家长
2. 初中生家长
3. 高中生家长
4. 对AI教育感兴趣的家长
5. 想提升亲子关系的家长
6. 想用AI辅导孩子学习的家长
7. 参加5天训练营的付费用户
8. 后续系列课/陪跑营用户

### 0.3 用户端核心体验

用户进入H5后，不是先看文章，而是先选择：

**“我现在遇到什么问题？”**

例如：

- 孩子拖拉磨蹭
- 孩子不爱学习
- 孩子数学不会
- 孩子作文写不出来
- 孩子英语不敢开口
- 孩子沉迷手机
- 孩子一写作业就吵架
- 孩子没有目标
- 家长忍不住吼孩子
- 高中选科不知道怎么选

点进去以后，系统给出：

1. 问题本质
2. 常见误区
3. 正确原则
4. 三步/五步SOP
5. AI提示词
6. 示例回答
7. 家长可直接说的话
8. 孩子行动任务
9. 观察指标
10. 相关问题推荐

# 5. 具体问题页模板示例

## 5.1 示例：孩子拖拉磨蹭怎么办？

### 适用年龄段

小学、初中

### 典型表现

- 写作业前一直准备；
- 写几分钟就停下来；
- 一会儿喝水，一会儿上厕所；
- 总说“等一下”；
- 家长越催越慢；
- 每天作业拖到很晚。

### 问题本质

拖拉不一定是懒，常见原因包括：

1. 任务太大，不知道从哪里开始；
2. 害怕做错，所以迟迟不开始；
3. 注意力容易被干扰；
4. 缺少时间感；
5. 家长长期催促，孩子形成被动模式。

### 家长常见错误做法

- 一直催；
- 大声吼；
- 用“你怎么这么懒”贴标签；
- 直接替孩子安排所有步骤；
- 一边陪一边批评。

### 正确处理原则

1. 先降低启动难度；
2. 再拆小任务；
3. 用可视化时间帮助孩子建立时间感；
4. 家长从催促者变成流程设计者；
5. 复盘时只谈方法，不贴标签。

### 三步SOP

#### 第一步：共情并命名问题

家长说：

> 我发现你不是不想写，而是每次开始的时候会有点困难。我们今天先不追求快，先练习怎么开始。

#### 第二步：把任务拆到足够小

例如：

- 第1步：拿出数学作业；
- 第2步：只看第一题；
- 第3步：圈出题目条件；
- 第4步：写出第一步思路；
- 第5步：完成前3题后休息2分钟。

#### 第三步：完成后复盘

问孩子：

- 今天哪一步最难开始？
- 哪一步开始后其实没那么难？
- 明天我们可以怎么让开始变容易？

### AI提示词

```text
请你扮演一位家庭教育指导师，帮我分析孩子拖拉磨蹭的问题。

孩子情况：
【填写年龄、年级、具体表现】

请你输出：
1. 孩子拖拉背后的3-5种可能原因；
2. 家长最容易犯的错误；
3. 适合今天晚上执行的3步解决方案；
4. 一段家长可以直接对孩子说的话；
5. 一个不超过30分钟的行动计划。

要求：
语气温和，不指责孩子；方法具体可执行。
```

### 家长可用话术

> 我看到你现在还没有开始写作业，我不是想骂你。我想和你一起找一个更容易开始的方法。我们先不管全部作业，只从第一小步开始，好不好？

### 孩子小任务

今天只练习一件事：

> 开始前先说出“我要做的第一小步是什么”。

### 观察指标

- 是否能在5分钟内启动；
- 是否减少家长催促次数；
- 是否能独立说出第一步；
- 作业总时长是否缩短；
- 冲突次数是否下降。

## 5.2 示例：初中数学不会做怎么办？

### 适用年龄段

初中

### 典型表现

- 看到题目不知道从哪下手；
- 会听课但不会做题；
- 应用题不会列式；
- 几何证明不会写过程；
- 错题反复错。

### 问题本质

初中数学困难通常不是“笨”，而是以下能力没有建立：

1. 审题能力；
2. 条件提取能力；
3. 数量关系建模能力；
4. 图形转化能力；
5. 题型归纳能力。

### 家长常见错误做法

- 直接讲答案；
- 一道题讲很多遍但孩子没参与；
- 责怪孩子不动脑；
- 只刷题不复盘；
- 不区分“不会读题”和“不会计算”。

### 五步SOP

1. 让孩子先说：这题问什么？
2. 圈出已知条件；
3. 问：这些条件之间有什么关系？
4. 让AI一步步引导，不直接给答案；
5. 做完后让孩子复述解题思路。

### AI提示词

```text
请你扮演一位初中数学老师，用苏格拉底提问法辅导孩子做题。

要求：
1. 不要直接给答案；
2. 每次只问一个问题；
3. 引导孩子先读题、找条件、找关系；
4. 如果孩子答错，请用提示引导，不要批评；
5. 最后让孩子用自己的话复述解题思路。

题目如下：
【粘贴题目】
```

### 家长可用话术

> 这道题我们先不急着算。你先告诉我：题目到底要求什么？已知条件有哪些？我们把条件圈出来，再一步一步看。

### 孩子小任务

每道不会的题必须先完成：

- 写出“求什么”；
- 圈出“已知条件”；
- 写出“我卡在哪一步”。

### 观察指标

- 是否能独立读题；
- 是否能圈出条件；
- 是否能说出卡点；
- 是否减少空白题；
- 是否能复述思路。

# 8. 产品金句

- 家长不需要更多道理，家长需要今晚就能用的方法。
- AI不是替代父母，而是让父母变得更会教育。
- 这个系统不是知识库，而是家庭教育问题解决中心。
- 用户不是来学习AI的，用户是来解决孩子问题的。
- 从场景进入，从问题定位，到SOP执行，再到AI提示词复制。
- 未来这个系统可以从内容库升级为AI家庭教育Agent。
