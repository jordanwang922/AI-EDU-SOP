# AI 亲子教育 SOP · 移动端 H5 设计风格说明（design.md）

> 定位：**小清新、透气、柔和**，面向以妈妈为主的家长群体；专业可信但不「公文气」、不「冷冰冰 SaaS」。  
> 本文与 Figma 文件 `AI亲子教育SOP · H5设计规范 v1` 中的 Variables 同步；开发以本文 Hex + Figma Token 为准。

---

## 1. 风格关键词（对内对齐）

| 关键词 | 含义 |
| --- | --- |
| **小清新** | 浅色大块留白、低饱和配色、轻盈不要用厚重色块压屏 |
| **柔和母性审美** | 偏暖中性灰文字、避免纯冷灰 (#64748B 一类)；点缀色用 **豆沙绿 / 雾粉 / 薄荷** 而非刺眼荧光 |
| **可信教育产品** | 主色仍要有「稳定感」，用 **鼠尾草绿 Sage** 作主品牌色，而不是幼幼粉铺满屏 |
| **耐看** | 参考近年潘通趋势里的 **雾粉 pastel、燕麦米白、透气浅绿**，避免高饱和「警戒橙」型 CTA（易显急躁、俗气） |

---

## 2. 色彩趋势参考（为何这样配）

以下仅为**灵感来源**，便于评审与迭代；**执行以色板表格为准**。

1. **Pantone 近年方向**  
   - **Cloud Dancer** 一类「透气浅色」作心智基底：大面积留白、柔雾感，减轻信息焦虑（适合教育 / 家庭场景）。  
   - **Powdered Pastels（雾粉 pastel）**：浅杏、雾紫、淡黄绿叠搭，强调温和而非艳丽。

2. **母婴 / 亲子类界面常见共识（国内站酷等案例归纳）**  
   - 主色多用 **低饱和暖色 + 浅绿（健康、成长）**，少用高饱和纯粉（易疲劳）。  
   - **圆角、大间距、扁平清爽图标**，传递「安全、轻松」。

3. **通用 pastel UI 实践**  
   - Pastel 做背景与区块；**正文用偏暖的深灰棕**，保证长时间阅读舒适。  
   - **强调色控制在 1～2 个**，其余用中性色过渡。

## 3. 品牌色板（执行规范）

### 3.1 背景与表面

| Token | Hex | 说明 |
| --- | --- | --- |
| `color/bg/page` | `#FAF7F4` | **燕麦米白**：页面底色，比纯白更暖，减轻刺眼感 |
| `color/bg/elevated` | `#FFFFFF` | 卡片 / 浮层 |
| `color/bg/subtle` | `#F3EEE8` | 次级分区背景（可选） |

### 3.2 文字（暖灰体系，避免冷 Slate）

| Token | Hex | 说明 |
| --- | --- | --- |
| `color/text/primary` | `#3D3835` | 标题与正文主色 |
| `color/text/secondary` | `#7A7269` | 次级说明 |
| `color/text/tertiary` | `#A89F94` | 辅助 / 禁用感 |

### 3.3 品牌与操作（小清新主副色）

| Token | Hex | 说明 |
| --- | --- | --- |
| `color/brand/primary` | `#8FAE9D` | **鼠尾草绿 Sage**：主按钮、选中、关键链接 —— 清新、稳定、偏「成长感」 |
| `color/brand/primary-pressed` | `#6F927F` | 按下 / 深一点的状态 |
| `color/action/cta` | `#E5989B` | **雾粉珊瑚**：强操作（如「复制提示词」）—— 柔和但不荧光，符合女性向审美 |
| `color/action/cta-pressed` | `#CF7F84` | CTA 按下 |

> 若需第三点缀色（标签、话题）：`#B8C5E0` **雾紫丁香**（仅小面积使用）。

### 3.4 边框与语义色（软化处理）

| Token | Hex | 说明 |
| --- | --- | --- |
| `color/border/subtle` | `#E8E4DF` | 暖灰边框 |
| `color/semantic/success` | `#89B89A` | 成功（柔和绿） |
| `color/semantic/warning` | `#D4A574` | 警告（柔琥珀，避免刺眼黄） |
| `color/semantic/error` | `#D48989` | 错误（柔玫瑰红，避免正红） |

### 3.5 与旧版色板的差异（说明给团队）

| 旧方案 | 调整原因 |
| --- | --- |
| Sky 蓝 `#0284C7` + 橙 CTA `#EA580C` | 偏「工具 / 政务」感，妈妈向小清新诉求下易显硬、偏男性向工具审美 |
| 冷灰文本 Slate | 不如暖灰亲和，与燕麦底色也更难融合 |

## 4. 布局与组件气质

- **圆角**：卡片 / 按钮用大圆角（与 Figma `radius/md`、`radius/lg` 一致），避免尖锐矩形。  
- **留白**：列表与段落 **偏大间距**，符合碎片化阅读与「减压」预期。  
- **图标**：线性或轻填色，避免厚重拟物。  
- **插画（若有）**：轻手绘 / 扁平，避免低幼夸张卡通。  

---

## 5. 无障碍与实现注意

- Pastel 背景上务必用 **`color/text/primary` 暖深灰** 保证可读性；雾粉按钮若配白字，需在实现侧核对对比度，不够则改为 **深字 + 浅粉底** 按钮样式。  
- 开发可用 CSS 变量与 Figma Variables **同名映射**，便于二次迭代。
