# AI 亲子教育 SOP · 移动端 H5 设计风格说明（design.md）

> 版本：**1.2**  
> 定位：**用户定制六色盘** — 天蓝 · 薄荷浴 · 奶油杏 · 柔珊瑚 · 浅鼠尾草 · 森绿字色；整体 **小清新、温柔、偏妈妈审美**，带一点高级感 pastel，又不像灰浊莫兰迪那样闷。  
> 本文与 Figma 文件 `AI亲子教育SOP · H5设计规范 v1` 中的 Variables 同步；开发以本文 Hex + Figma Token 为准。

---

## 1. 风格关键词（对内对齐）

| 关键词 | 含义 |
| --- | --- |
| **定制小清新** | 以 **浅薄荷 `#DAEBE3`** 作页面氛围底，搭配 **天蓝 `#99CDD8`** 作主交互色，整体清透 |
| **柔珊瑚强调** | **`#F3C3B2`** 作强操作（复制提示词等）；其上文字优先用 **`#657166` 森绿** 保证对比（慎用纯白字） |
| **森绿字色** | **`#657166`** 作为主文字与图标深色，统一且不发黑发闷 |
| **版本演进** | v1.2 为用户给定参考色卡；v1.1（Teal+玫粉）及更早燕麦系可由本文表格一次性替换 |

## 4. 布局与组件气质

- **圆角**：卡片 / 按钮保持大圆角；**天蓝按钮**与 **珊瑚按钮**并列时，珊瑚承担「唯一最强动作」。  
- **留白**：薄荷底上出现 **白卡片**，层次清晰。  
- **插画 / 摄影**：可大胆使用 **天蓝与奶油杏** 做渐变氛围（极淡即可），避免厚重灰滤镜。  

---

## 5. 无障碍与实现注意

- **`#F3C3B2` 上请勿默认白字**：优先 **森绿字 `#657166`**；若需反白，必须跑对比度检测。  
- **`#DAEBE3` 上正文** 使用 **`#657166`** 一般可读性良好。
