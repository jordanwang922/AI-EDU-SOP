## A. 家长端（当前编号）

**无开屏**：打开 H5 **第一个画面即首页**（搜索 + 大类）。

### 主路径（① → ② → ③）

| # | Frame | 用户看到什么 |
| --- | --- | --- |
| **01** | **`01 · 首页 · 大类与搜索`** | 顶部 **关键词搜索框**（示例占位「拖拉」等）；热门标签 chip；**一级大类/场景**列表（卡片入口） |
| **02** | **`02 · 问题列表 · 二级即具体问题`** | 返回；路径「首页 › 某大类」；**列表每一项是一条具体问题**；可用分组标题归类（非单独页面） |
| **03** | **`03 · 方案详情`** | 返回；标题；面包屑；标签 chip；各内容区块（表现、本质、怎么做、步骤、话术、提示词等）；底部操作按钮 |

### 搜索支线

| # | Frame | 说明 |
| --- | --- | --- |
| **04** | **`04 · 搜索`** | 搜索框 + 「猜你想搜」标签词 |
| **05** | **`05 · 搜索结果`** | 结果列表 |
| **06** | **`06 · 搜索无结果`** | 提示与返回引导 |

### 状态

| # | Frame | 说明 |
| --- | --- | --- |
| **07** | **`07 · 复制成功提示`** | 复制反馈 |
| **08** | **`08 · 加载中`** | 骨架 |
| **09** | **`09 · 网络异常`** | 重试 |
