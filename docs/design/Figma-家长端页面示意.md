## A. 家长端（主路径与检索）

| Frame | 场景说明 |
| --- | --- |
| **00 · 开屏 / 启动** | 品牌标题、价值一句话、奶油杏英雄区 + 天蓝主按钮「进入树状目录」、脚注 1.0 无登录。 |
| **01 · 首页（大类）** | 主标题、快捷胶囊、「为你推荐」、8 条一级入口、cream 提示条。 |
| **02 · 二级 · 学习问题** | 面包屑、PRD 4.1 五个二级分类卡片。 |
| **03 · 问题列表 · 拖拉磨蹭** | 面包屑、统计、10 条叶子标题 + hint。 |
| **04 · 详情 · 词条模板示意** | 精简版叶子页：标签、本质/SOP、预览条、天蓝 + **柔珊瑚复制 CTA（森绿字）**。 |
| **08 · 详情 · 完整词条（PRD 全字段示意）** | **长页**：典型表现 → 本质 → 误区 → 原则 → SOP → 提示词 → 示例 → 话术 → 任务 → 指标 → 复盘 → 相关推荐 → 双按钮；脚注 1.0 无终端 AI。 |
| **05 · 搜索** | 顶栏、搜索框占位、热门 chip；标注 1.0 搜索为可选能力。 |
| **06 · 搜索结果** | 若干结果卡片（面包屑路径 + 标题）、加载更多。 |
| **07 · 搜索无结果** | 提示 + chip（换关键词 / 回首页 / 看热门）。 |
| **09 · 交互反馈 · 复制成功 Toast** | 示意详情区内嵌 **深色底 Toast**（可复制为 cream 弱提示）。 |
| **10 · 二级 · 亲子沟通** | 第二条路径：PRD 4.2 四个二级主题卡片。 |
| **11 · 问题列表 · 孩子顶嘴** | 9 条具体问题标题（对齐 PRD 4.2.2）。 |
| **12 · 加载中 · Skeleton** | 列表骨架屏占位条。 |
| **13 · 异常 · 网络失败** | 文案 + 天蓝「重试」。 |

手机框宽度 **390px**；主底 **薄荷浴 `#DAEBE3`**，主交互 **天蓝**，强 CTA 底 **柔珊瑚 + 森绿字**。

## B. 运营后台（内容生产闭环）

| Frame | 场景说明 |
| --- | --- |
| **B01 · 后台登录** | 居中卡片：账号密码（与学员端分离）。 |
| **B02 · 仪表盘** | 顶栏 + 发布数/草稿/待审核卡片 + 表格占位（热门、复制次数等可对齐 PRD 2.2.1）。 |
| **B03 · 分类管理 · 树** | 左树右详情：排序、年龄段、前台展示、保存（柔珊瑚主按钮示例）。 |
| **B04 · 词条列表 · 筛选** | 筛选条 + 表格占位 +「新建词条」。 |
| **B05 · 词条编辑 · 表单** | **PRD 全字段**左右分栏：左表单 / 右预览占位；底栏 草稿 / 提交审核 / 发布。 |
| **B06 · AI 生成词条 · 弹层** | 选题输入 → **生成草稿**（天蓝）；脚注：后台 AI 必备，家长端仅已发布内容。 |

后台画布 **第二行**：`B05`、`B06` 与第一行错开，避免与 `B01` 重叠。
