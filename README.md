# AI 亲子教育 SOP 系统

面向家长的家庭教育场景：**场景检索 → 结构化 SOP 与 AI 提示词 → 复制即用**。后台支持分类与词条管理，并支持 **AI 辅助生成内容、审核后入库发布**。前期交付形态以 **H5** 为主，架构预留 **小程序 / App** 共用同一后端能力。

详细产品与架构说明见 **`docs/README.md`**。

## 仓库结构（初始化阶段）

```
docs/           # 产品 / 架构 / PRD / 开发日志与交接日志（单一事实来源）
```

应用代码目录将在确定技术栈后补充。

## 协作约定

- 每次开发结束请在 **`docs/development/开发日志.md`** 追加记录。
- 跨 AI / 跨人接手请更新 **`docs/development/交接日志.md`**。
- 日志文件超过 **500 行**按 `docs/meta/文档维护规范.md` 归档。
